NAME
	Parallel Mandelbrot

USAGE
	1. make -f makefile
	2. ./ParallelMandelbrot

DESCRIPTION
	By Default, this program will test the running time of generating a 640x480 Mandelbrot .PPM
	using various numbers of cores. It will start at 2 cores and ramp up to 8. The first step may
	take some time.



Compiled with GNU G++ 4.9.2 on OS X 10.9

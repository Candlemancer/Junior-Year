NAME
	Scheduling Framework (FIFO)

USAGE
	1. make -f makefile
	2. ./SchedulingFramework

DESCRIPTION
	Simulates a device running a number of computational tasks. More details are provided in the
	application itself.


Compiled with GNU G++ 4.9.2 on OS X 10.9
Compatible with any POSIX-compliant system
